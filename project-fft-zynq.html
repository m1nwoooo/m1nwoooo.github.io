Designed and implemented a 128-point FFT on a Xilinx Zynq-7000

📅 2023.10 - 2023.12
👥 Individual Project
🎯 Academic Project

Xilinx Zynq-7000 SoC를 활용하여 128포인트 FFT를 하드웨어와 소프트웨어의 협력으로 구현한 프로젝트입니다.

ARM 프로세서와 FPGA를 결합한 Zynq의 특성을 활용하여, 계산 집약적인 부분은 FPGA의 하드웨어 가속기로 처리하고, 제어 로직은 ARM 프로세서에서 담당하는 하이브리드 아키텍처를 구현했습니다.

AXI4 인터페이스를 통한 효율적인 데이터 통신과 DMA를 활용한 고속 메모리 접근을 통해 최적화된 성능을 달성했습니다.

사용 기술
Xilinx Zynq-7000 Verilog HDL AXI4 ARM Cortex-A9 C/C++ Vivado
