Development of a Helmet-Mounted System

📅 2023.05 - 2023.08
👥 Team Project (4명)
🎯 Capstone Design

산업 현장이나 위험 지역에서 작업자의 안전을 보장하기 위한 스마트 헬멧 시스템을 개발한 프로젝트입니다.

다양한 센서(가스 센서, 온도 센서, 자이로스코프)를 통해 작업 환경을 실시간으로 모니터링하고, 위험 상황 발생 시 즉시 경고 신호를 발생시키는 시스템을 구현했습니다. 또한 GPS와 무선 통신을 통해 관제센터와 실시간으로 위치 및 상태 정보를 공유할 수 있도록 했습니다.

웨어러블 디바이스의 특성상 경량화와 저전력 설계에 중점을 두었으며, 실제 작업 환경에서의 테스트를 통해 시스템의 신뢰성을 검증했습니다.

사용 기술
Arduino ESP32 Sensor Integration IoT C/C++ PCB Design
